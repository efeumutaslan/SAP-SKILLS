# SAP Terminology Quick Reference

## Correct Casing (Always Use These)

| Correct | Wrong | Notes |
|---------|-------|-------|
| ABAP | Abap, abap | Always uppercase |
| BAdI | BADI, Badi | Business Add-In |
| BTP | Btp | Business Technology Platform |
| CAP | Cap | Cloud Application Programming Model |
| CDS | Cds | Core Data Services |
| EML | Eml | Entity Manipulation Language |
| Fiori | FIORI, fiori | SAP Fiori (capital F only) |
| HANA | Hana | High-performance ANalytic Appliance |
| OData | Odata, ODATA | Open Data Protocol |
| RAP | Rap | RESTful Application Programming Model |
| SAPUI5 | SapUI5, sapui5 | SAP UI5 framework |
| S/4HANA | S4HANA, S/4 Hana | Always with slash and uppercase |
| XSUAA | Xsuaa | XS Advanced UAA service |
